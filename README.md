# 🌤️ Weather Dashboard

A simple and responsive **Weather Dashboard Web App** that displays real-time weather data including current conditions, hourly forecast, and 7-day forecast.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature display
- ☁️ Weather condition with icons
- 📊 Detailed weather info:
  - Feels like temperature
  - Humidity
  - Wind speed
  - Pressure
- ⏰ Hourly forecast (next 24 hours)
- 📅 7-day weather forecast
- ⚡ Fast and lightweight UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Open-Meteo API (Weather Data)
- Open-Meteo Geocoding API (City search)
